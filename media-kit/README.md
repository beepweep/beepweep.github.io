# BeepWeep Ventures Media Kit

This folder contains official BeepWeep Ventures brand assets for media, press, and partnership use.

## Logo Variations

### Wordmark (Horizontal)
Use for headers, banners, and wide format applications.

**Available Formats:**
- `beepweep-wordmark-horizontal-color-light.svg` - Full color on light backgrounds
- `beepweep-wordmark-horizontal-color-dark.svg` - Full color on dark backgrounds  
- `beepweep-wordmark-horizontal-black.svg` - Black version for light backgrounds
- `beepweep-wordmark-horizontal-white.svg` - White version for dark backgrounds

### Icon (Square)
Use for social media profiles, app icons, and square format applications.

**Available Formats:**
- `beepweep-icon-square-color-light.svg` - Full color on light backgrounds
- `beepweep-icon-square-color-dark.svg` - Full color on dark backgrounds
- `beepweep-icon-square-black.svg` - Black version for light backgrounds
- `beepweep-icon-square-white.svg` - White version for dark backgrounds

## File Formats

### SVG (Vector)
- Location: `logos/svg/`
- Scalable to any size without quality loss
- Best for: Print, large displays, web

### PNG (Raster)
- Locations: 
  - `logos/png/512px/` - Standard web use
  - `logos/png/1024px/` - High resolution web/print
  - `logos/png/2048px/` - Ultra high resolution print
- Pre-rendered at multiple sizes
- Best for: Quick implementation, social media

## Brand Colors

- **International Orange**: `#ff4d00` (RGB: 255, 77, 0)
- **Hyper Blue**: `#0055ff` (RGB: 0, 85, 255)
- **Black**: `#000000`
- **White**: `#ffffff`

## Usage Guidelines

1. **Minimum Size**: Never display the wordmark smaller than 120px wide
2. **Clear Space**: Maintain clear space equal to the height of the "B" on all sides
3. **Background**: Use appropriate color variations for contrast
4. **Do Not**: 
   - Alter colors
   - Rotate or distort
   - Add effects (shadows, gradients, etc.)
   - Place on busy backgrounds
